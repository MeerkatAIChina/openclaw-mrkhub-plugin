## [0.1.0] - 2026-05-22

### Added

- OpenClaw 插件：通过 `/mrkhub` 从 GitHub 检索并安装 Meerkat skills
- 关键词匹配搜索与确定性安装到 `~/.agents/skills/<name>/`
- 多轮对话指代（基于会话内上次搜索结果）
- 可选 Agent 工具：`mrkhub_search`、`mrkhub_install`
- 支持配置 GitHub 仓库列表、安装目录、token 与默认 ref
